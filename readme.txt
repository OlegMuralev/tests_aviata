Setup:
1) В нашем виртуальном окружении необходимо установить библиотеку Selenium:
pip install selenium==4.4.3
2) Далее скачать драйвер Chromium для браузера. Я использовал версию 106.
Скачать нужную версию можно с официального сайта по ссылке https://sites.google.com/chromium.org/driver/downloads

Запуск:
python test_popular_directions.py
